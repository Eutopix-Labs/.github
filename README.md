# Eutopix Labs                                                                          
                                                                                          
  An independent European studio in Warszawa, building privacy-first                      
  consumer apps and AI products that respect the people who use them.
                                                                                          
  No advertising. No data brokers. No behavioural profiling.                              
  Hosted in Europe, on green-energy infrastructure, with open-source                      
  models wherever we can run them.                                                        
                                                                                          
  → [eutopixlabs.io](https://eutopixlabs.io)                                              
                                                                                          
  ## What we're working on                                                                
  
  Pre-launch as of Spring 2026. Our first app is a privacy-first                          
  nutrition tool that combines barcode scanning, calorie tracking,
  and AI photo-to-nutrition — on-device, in one place. We're finishing                    
  it now and we're in no particular hurry. Software made slowly tends                     
  to age better.                                                                          
                                                                                          
  ## Principles                                                                           
                                                                                          
  - **European hosting, on principle.** EU vendors first. Where that                      
    isn't possible yet, we say so plainly.
  - **Open-source AI, green hosting.** Open-source models on green-energy                 
    EU infrastructure. Closed APIs only when nothing else fits, and only                  
    when we name them.                                                                    
  - **No tracking of any sort.** Our apps are paid for by the people                      
    who use them. That's the whole arrangement.
